# address-catalog

Este projeto foi desenvolvido utilizando Vue 3 e Vite.

É um catálogo de endereços simples com um banco de dados no browser em SQLite.

É possível criar um registro a partir do botão "+", podemos baixar toda a lista criada em CSV no botão de download, podemos apagar um registro da lista clicando no ícone de lixeira e também podemos alterar um pedido para o status de concluído clicando sobre o ícone de status.

## Configuração do projeto

- Instalação necessária do Node.js 18

```sh
npm install
```

### Rodar em modo de desenvolvimento

```sh
npm run dev
```

### Type-Check, Compilação e Minificação para produção

```sh
npm run build
```

### Rodar Unit Tests com [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```

### Lint com [ESLint](https://eslint.org/)

```sh
npm run lint
```
